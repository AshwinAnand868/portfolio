Payroll App

Assembly line simulator

Using C++

Assembly line simulator

This is a console application which simulates the process of an automated assembly line. The line consists of many stations and each station processes and completes the customers' orders depending upon if it has the specified item in stock, otherwise moves the item to the next station for further processing until it reaches to the end.
